# Q: Audit and fix battery sizing's monotonic-search assumption. Induced power can scale superlinearly with battery mass, so exponential upper-bound probes can skip a feasible capacity interval.

## Answer

Expanded from original query via graph vocabulary: [battery, sizing, capacity, mass, energy, induced, feasible, search, reserve, mtow, power, minimum]. The old exponential upper probes and global binary search assumed monotone feasibility and could skip a bounded middle interval. The implementation now scans the full physical capacity domain from the positive mass-valid minimum to the MTOW-derived maximum at the requested tolerance until it observes the first feasible transition, refines both bounds of that contiguous feasible interval, treats only capacity-dependent reserve failures as searchable, and reports unrelated estimator blockers by code. Battery-mass feedback remains active for every probe. battery-sizing-report.v2 exposes the minimum, verified upper bound, MTOW capacity, resolution, and fail-closed margin recommendations; recommendations beyond the upper bound are null with a reason. Regression coverage uses a narrow false-true-false window generated by fixed energy plus a sixth-power induced mass term.
